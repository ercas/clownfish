About Clownfish

Jump to: security tools

        Clownfish is an ephemeral, free and open source (FOSS) file conversion service acting as a frontend for many FOSS tools that can convert files from one format to another.
        Clownfish can be deployed quickly as a Docker container or in a virtual machine to serve file conversion needs for people without access to a Unix-like operating system. It can also be deployed on a barebones Linux install, but in this case, it would no longer be ephemeral.
        The source code for Clownfish is available on GitHub under an MIT License. This means you can do whatever you want with the source code as long as you include the license in derivative works and don't hold the author liable.

Clownfish is not secure!

        Clownfish is currently an insecure service; any file that you send to or receive from Clownfish can be intercepted by a third party. Do not use Clownfish over the Internet for any sensitive file conversions; instead, run it in a container or virtual machine on your home network and make sure that there are no bad actors on your network and that Clownfish is not connected to the Internet.

Tools used

All tools used in Clownfish are free and open source.

  • ImageMagick, a universal image editor with a command line interface that can convert almost any image file into an image of almost any other format
  • FFmpeg, the ImageMagick of audio and video files
  • LibreOffice, an office suite that can convert between OpenDocument and Microsoft Office formats and generate PDF files from those formats
  • Binwalk, a firmware analysis tool
  • Foremost, a fast, public domain data carving tool written by the U.S. Air Force and Naval Postgraduate School
