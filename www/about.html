Clownfish

a universal file conversion service

        Clownfish is an ephemeral, free and open source (FOSS) file conversion service, acting as a web-based frontend for many FOSS tools that can convert files from one format to another.

        Clownfish can be deployed quickly as a Docker container or in a virtual machine to serve file conversion needs for people without access to a Unix-like operating system. It can also be deployed on a barebones Linux install, but in this case, it would no longer be ephemeral.
